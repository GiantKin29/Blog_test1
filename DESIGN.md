# ブログデザイン仕様書

## 参考にした日本の人気ブログ・トレンド

| テーマ / サイト | 特徴 |
|---|---|
| **SWELL** | 国内人気No.1 WordPress テーマ。シンプル・高速・白ベース・丸みのあるUI |
| **AFFINGER6** | 収益化特化。ヒーロー画像・カード型・視認性重視 |
| **JIN:R** | 23種プリセット。カード型記事一覧・カテゴリナビ |
| **2026年Webデザイントレンド** | Bento UI、ゆとりある余白、ミュートカラー、安心感 |
| **金融サイトのトレンド** | 信頼感のある紺・青系、緑（成長）、クリーンなタイポグラフィ |

## 4. ヘッダー

- **背景**: 白 `#FFFFFF`
- **ボトムボーダー**: `3px solid #1D4ED8`（青ライン）
- **ドロップシャドウ**: `0 2px 12px rgba(0,0,0,0.07)`
- **sticky**: スクロール固定
- **ロゴ**: サイト名（青 + 太字）
- **ナビ**: テキストリンク、ホバー時青色

## 5. ヒーローセクション

- **背景**: 青グラデーション `#1D4ED8 → #2563EB → #3B82F6`
- **テキスト**: 白
- **レイアウト**: 中央揃え
- **構成**:
  - ラベル（pill形式）
  - メインキャッチコピー（大きめ、太字）
  - サブコピー
  - CTAボタン（白背景・青テキスト）

## 6. カテゴリナビゲーション

## 7. 記事カード

```
┌─────────────────────────┐
│  [サムネイル 16:9]       │ ← カテゴリ別グラデーション（画像なし時）
│  [カテゴリバッジ]        │ ← 左上オーバーレイ
├─────────────────────────┤
│  記事タイトル            │ ← bold, 最大3行
│  2024.01.01  · 5分       │ ← 日付 + 読了時間
└─────────────────────────┘
```

- **角丸**: `14px`
- **シャドウ**: `0 2px 10px rgba(0,0,0,0.06)`
- **ホバー**: `translateY(-4px)` + シャドウ強化 + ボーダー着色
- **グリッド**: 2列（PC） / 1列（スマホ）

## 11. モバイル対応

| ブレークポイント | 変化 |
|---|---|
| `< 1024px` | 2カラム → 1カラム（サイドバーを下に移動） |
| `< 768px` | カードグリッド 2列 → 1列 |
| `< 640px` | ヒーロー文字サイズ縮小 |

---

## 12. アニメーション・UX

- カードホバー: `transform: translateY(-4px)` + `box-shadow` 変化（0.2s ease）
- カテゴリナビ: `translateY(-2px)` ホバー
- CTAボタン: `translateY(-2px)` + `box-shadow` 強化
- スクロールバー: 青アクセント
